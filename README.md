🚕 Taxi Sharing Backend – Django + DRF + PostgreSQL

A robust backend system designed for a taxi-sharing platform, built with Django, Django REST Framework, and PostgreSQL. This project provides secure and scalable APIs for managing users, rides, drivers, payments, and location-based ride matching.

The backend includes features such as:

🔐 User authentication & authorization

🚖 Ride creation, booking, and sharing

💳 Payment handling structure

👤 Driver & passenger management

📡 RESTful API endpoints ready for mobile or frontend integration

🏗️ Clean, modular architecture (users, rides, payments apps)

🗂️ Environment-safe project structure with .env support
